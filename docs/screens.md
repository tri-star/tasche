## 各画面詳細

### SCR-01: ログイン画面

**URL**: `/login`

**概要**: Auth0を使用したソーシャルログイン画面

**構成要素**:
| 要素ID | 要素名              | 種類     | 説明                                 |
| ------ | ------------------- | -------- | ------------------------------------ |
| L-01   | ロゴ                | 画像     | Tascheロゴ                           |
| L-02   | アプリ説明          | テキスト | 簡単なサービス説明                   |
| L-03   | Googleログインボタン | ボタン   | Google OAuth でログイン              |
| L-04   | GitHubログインボタン | ボタン   | GitHub OAuth でログイン              |

**画面遷移**:
- ログイン成功 → SCR-02（ダッシュボード）

---

### SCR-02: ダッシュボード

**URL**: `/`（ルート）

**概要**: メイン画面。今週の達成状況確認と実績入力を行う

**構成要素**:

#### ヘッダー
| 要素ID | 要素名           | 種類     | 説明                                 |
| ------ | ---------------- | -------- | ------------------------------------ |
| D-01   | ロゴ             | 画像     | クリックでダッシュボードに戻る       |
| D-02   | ナビゲーション   | リンク群 | タスク管理、設定へのリンク           |
| D-03   | ダークモード切替 | トグル   | ライト/ダークモード切替              |
| D-04   | ユーザーメニュー | ドロップダウン | ログアウト等                    |

#### 今日の目標ウィジェット
| 要素ID | 要素名             | 種類     | 説明                                 |
| ------ | ------------------ | -------- | ------------------------------------ |
| D-10   | 今日の日付         | テキスト | 「2024年1月15日（月）」形式          |
| D-11   | タスク目標リスト   | リスト   | タスク名と目標ユニット数の一覧       |

#### 実績登録ウィジェット
| 要素ID | 要素名                   | 種類         | 説明                               |
| ------ | ------------------------ | ------------ | ---------------------------------- |
| D-20   | 曜日セレクタ             | タブ/ボタン群 | 今週の曜日を選択                  |
| D-21   | タスク選択コンボボックス | コンボボックス | フィルター付きタスク選択         |
| D-22   | ユニット数入力           | スピンボタン | 0.1単位で増減可能な数値入力       |
| D-23   | 記録ボタン               | ボタン       | 実績を記録                         |

#### 週間達成状況マトリックス
| 要素ID | 要素名           | 種類   | 説明                                   |
| ------ | ---------------- | ------ | -------------------------------------- |
| D-30   | マトリックス表   | テーブル | 行=タスク、列=曜日、セル=消化率      |
| D-31   | 消化率セル       | セル   | 消化率を数値と色（赤〜緑）で表示       |
| D-32   | 合計行           | 行     | 各曜日の合計消化率（オプション）       |

#### フローティングアクション
| 要素ID | 要素名             | 種類   | 説明                                 |
| ------ | ------------------ | ------ | ------------------------------------ |
| D-40   | 目標設定ボタン     | FAB    | 目標設定画面を開く                   |

**画面遷移**:
- 目標設定ボタン → SCR-03（目標設定画面）
- タスク管理リンク → SCR-04（タスク管理画面）
- 設定リンク → SCR-05（設定画面）

### SCR-03: 目標設定画面

**URL**: `/goals`

**概要**: 週の目標をウィザード形式（ステップ式）で設定する画面

#### Step 1: ユニット時間選択
| 要素ID | 要素名             | 種類       | 説明                               |
| ------ | ------------------ | ---------- | ---------------------------------- |
| W1-01  | ステップインジケータ | UI       | 現在のステップを表示               |
| W1-02  | 説明テキスト       | テキスト   | 「1ユニットの時間を選んでください」 |
| W1-03  | 時間選択ボタン群   | ラジオボタン | 10分/30分/1時間/2時間            |
| W1-04  | 次へボタン         | ボタン     | Step 2へ進む                       |

#### Step 2: タスク選択
| 要素ID | 要素名               | 種類         | 説明                             |
| ------ | -------------------- | ------------ | -------------------------------- |
| W2-01  | ステップインジケータ | UI           | 現在のステップを表示             |
| W2-02  | 説明テキスト         | テキスト     | 「今週取り組むタスクを選んでください」 |
| W2-03  | 既存タスク一覧       | チェックリスト | 既存タスクから選択              |
| W2-04  | 新規タスク追加       | 入力+ボタン  | 新しいタスクを追加               |
| W2-05  | 戻るボタン           | ボタン       | Step 1へ戻る                     |
| W2-06  | 次へボタン           | ボタン       | Step 3へ進む                     |

#### Step 3: 曜日別目標設定
| 要素ID | 要素名               | 種類       | 説明                               |
| ------ | -------------------- | ---------- | ---------------------------------- |
| W3-01  | ステップインジケータ | UI         | 現在のステップを表示               |
| W3-02  | 説明テキスト         | テキスト   | 「各タスクの曜日ごとの目標を設定」 |
| W3-03  | 目標設定マトリックス | テーブル   | 行=タスク、列=曜日、セル=目標入力  |
| W3-04  | 戻るボタン           | ボタン     | Step 2へ戻る                       |
| W3-05  | 次へボタン           | ボタン     | Step 4へ進む                       |

#### Step 4: 確認
| 要素ID | 要素名               | 種類     | 説明                               |
| ------ | -------------------- | -------- | ---------------------------------- |
| W4-01  | ステップインジケータ | UI       | 現在のステップを表示               |
| W4-02  | 設定サマリー         | 表示     | ユニット時間、タスク、目標の一覧   |
| W4-03  | 戻るボタン           | ボタン   | Step 3へ戻る                       |
| W4-04  | 保存ボタン           | ボタン   | 設定を保存してダッシュボードへ     |

**画面遷移**:
- 保存ボタン → SCR-02（ダッシュボード）
- キャンセル → SCR-02（ダッシュボード）

### SCR-04: タスク管理画面

**URL**: `/tasks`

**概要**: タスクの一覧表示・編集・削除を行う画面

**構成要素**:
| 要素ID | 要素名           | 種類       | 説明                               |
| ------ | ---------------- | ---------- | ---------------------------------- |

## レスポンシブ対応

### ブレークポイント

| サイズ   | 幅         | 対象デバイス       |
| -------- | ---------- | ------------------ |
| sm       | < 640px    | モバイル           |
| md       | 640-1024px | タブレット         |
| lg       | > 1024px   | デスクトップ       |

### 各画面のレスポンシブ対応方針

| 画面               | モバイル対応方針                                   |
| ------------------ | -------------------------------------------------- |
| ダッシュボード     | ウィジェットを縦並びに配置、マトリックスは横スクロール |
| 目標設定ウィザード | 各ステップはフルスクリーン表示                     |
| タスク管理画面     | リスト形式でそのまま表示                           |
| 設定画面           | リスト形式でそのまま表示                           |

## ダークモード対応

すべての画面でダークモードに対応。

| 要素           | ライトモード | ダークモード |
| -------------- | ------------ | ------------ |
| 背景色         | 白系         | 黒/灰系      |
| テキスト色     | 黒系         | 白系         |
| アクセント色   | ブランドカラー | ブランドカラー |
| 消化率（0%）   | 赤           | 赤           |
| 消化率（100%） | 緑           | 緑           |
