# Git コミットに関するガイドライン

- 開発作業の1ステップが終わる都度関連するファイルをコミットするようにしてください。
  (コミットを見ることで、この作業でどのファイルがどのようにへんこうされたか分かるようにしたい)
- コミットメッセージは日本語で、変更内容の概要を記述してください。
  - ソースコード上には残せない「なぜこの変更を行ったのか」も含めてください。

- `.claude/agent-memory` フォルダ配下が更新されることがあります。これはエージェントが自身が実行したタスクに関するメモを長期記憶用に保存するものなので、
  これらが更新されていた場合は独立したコミットとしてコミットしてください。
